# ROADMAP.md — מסמך המשימות של החזון

זהו מסמך המשימות שכל סוכן AI עובד לפיו. כשאין משימה ספציפית — לוקחים את המשימה הפתוחה הראשונה לפי הסדר. אחרי כל משימה שהושלמה מעדכנים כאן ☐ → ✅.

**מקרא:** ✅ הושלם · 🔄 בתהליך · ☐ פתוח

## שלב 0 — תשתית פרויקט 🔄

- ✅ ריפו Git + דחיפה ל-GitHub‏ (https://github.com/avivit-viskin/vaddygo)
- ✅ מערך תיעוד מלא בריפו (README, ARCHITECTURE, AI_RULES, ROADMAP, PROJECT_LOG, DEPLOYMENT, AGENT_PROMPT)
- ✅ CI ב-GitHub Actions: בדיקות + build על כל push
- ☐ פריסה ל-Railway + הפקת דומיין (לפי DEPLOYMENT.md) — באחריות בעלת המוצר עם ליווי
- ☐ העלאת הבקאנד (ParentCommitteeAPI) ל-GitHub כריפו נפרד + פריסתו ל-Railway

## שלב 1 — תשתית פרונטאנד ☐

- ☐ מבנה תיקיות מלא: components / pages / services / hooks / styles
- ☐ `services/api.js` יחיד עם `REACT_APP_API_URL` (סוף לכתובות קשיחות)
- ☐ ערכת עיצוב: צבעים (סגול `#7C3AED`), טיפוגרפיה, RTL גלובלי
- ☐ רכיבי בסיס גנריים: Button, Input, Card, Modal, Table, Spinner, EmptyState, ErrorMessage
- ☐ React Router + שלד ניווט תחתון (Bottom Navigation)
